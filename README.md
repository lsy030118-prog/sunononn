# 절댓값을 포함한 일차부등식 탐구

이 앱은 고등학교 1학년 수준의 절댓값을 포함한 일차부등식 학습을 돕기 위해 만들어졌습니다.

## 사용 방법

1. 첫 번째 절댓값 내부에 일차식을 입력합니다.
2. 두 번째 절댓값 내부에 일차식을 입력합니다.
3. `+` 또는 `-` 연산자를 선택합니다.
4. 부등호 방향과 등호 포함 여부를 선택합니다.
5. 우변에 숫자 또는 식을 입력합니다.
6. `풀이 시작` 버튼을 누르면 구간별 선형식, 부등식 풀이, 그래프, 최종 해 및 수직선 표현이 나타납니다.

## 배포 안내

이 앱을 Streamlit Cloud에 배포하면 `*.streamlit.app` 형태의 URL로 공유할 수 있습니다.

예: `https://your-app-name.streamlit.app`

## 요구 사항

- Python
- streamlit
- sympy
- numpy
- matplotlib

설치:

```bash
pip install -r requirements.txt
```
